# RegistroViajero

Sitio público de [RegistroViajero](https://registroviajero.com), software SaaS para el cumplimiento del Real Decreto 933/2021 en alojamientos turísticos españoles. Check-in digital, envío automático de partes de viajeros a SES.HOSPEDAJES y sincronización de reservas vía iCal.

Sitio bilingüe (español e inglés), estático, generado con Astro y Tailwind CSS v4.
